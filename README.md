# AI Resume Scanner

AI-powered Resume Screening System built using Python, Pandas, NumPy, and PyPDF2.

## Features

* PDF Resume Scanning
* Skill Extraction
* ATS-style Resume Scoring
* Candidate Shortlisting
* CSV Report Generation

## Tech Stack

* Python
* Pandas
* NumPy
* PyPDF2

## Project Structure

resume_scanner_project/
│
├── resumes/
├── output/
├── scanner.py
├── requirements.txt
└── README.md

## How to Run

1. Add PDF resumes inside the resumes folder
2. Run scanner.py
3. View generated CSV results

## Installation

pip install pandas numpy PyPDF2
